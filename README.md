# 📊 InterviewAI


## 🎯 프로젝트 개요

**InterviewAI**  
Streamlit 기반의 면접 연습 웹 앱  
GPT API를 활용한 질문 및 답변 피드백 제공


## 🧠 핵심 기능

- 키워드 입력 → 면접 질문 자동 생성  
- 사용자 답변 입력 및 수정  
- GPT 기반 피드백 제공  
- 전체 답변에 대한 종합 평가 요약
